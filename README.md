# Naukma devops 

## To run this project run 

```bash
docker run --cpus="1.5" --memory="300m" -p 80:3000 kkpagaev/naukma-devops:latest
```

## To build this project run 

```bash
docker build -t kkpagaev/naukma-devops:latest .
```

## To push this project to docker hub run 

```bash
docker push kkpagaev/naukma-devops:latest
```
